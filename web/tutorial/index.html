Draggable Chatbox with Vue framework

Difficulty: 10/10

This is completed with CSS and JS. You still need only 1 html div as a location to attach your code in or you can attach it to the html body itself(not a good practice though)

Create a css and a js file to host these codes below.

One problem with this Draggable div is that I'm unable to do createElement like usual since there are too much render priority issues and some objects get renders before or after other object, making it impossible to incorporate all together. This is why I cheated a little bit and do normal code injection. :((

First, create all the necessary element of the audio with js, including their class and id with normal code injection instead of createElement method. Add the css in since some functionality also required the position:abosolute of css to work. Add them to the html div in another html file like the coming soon page

Basically, there are three main part for the Draggable Div: the header, the body and the text area

The header is the main draggable part, as well as containing the button for minimize function

The body is where the text input get display

The text area contain a text input box, a submit and a clear button

Second, add an initial position to the div so that it doesn't obstruct other content. Then add 3 functions to allow setting initial position, dragging the whole element, and minimize the element when needed

The function for setting initial position is self-explanatory. The function to drag the element will offset the top and the left position of the whole element to the cursor position. The minimize will activate when the user click on the minimize line on the header. It will hide the body and the text area and left only the header visible.

Third, Integrate some vue component in the same js file (the vue cdn is already incorporated in the first step)

The vue element is mainly for text storing from the text input area and display it in the body.Vue does provide a quicker way to work with displaying data but it takes 15 lessons of Vue to get to this point, so I'm not gonna explain everything that Vue does.

Finally, add them all together
